## Pomodoro Timer with Task List

This is a simple, single-page web app that combines a **Pomodoro timer** with a **task list** to help you stay focused and organized.

### Features

- **Pomodoro timer** with start, pause, and reset controls
- **Session types** (Pomodoro, Short Break, Long Break)
- **Task list** where you can add, complete, and delete tasks
- **Automatic saving** of tasks in your browser (localStorage)
- **Responsive, modern UI** that works on desktop and mobile

### Getting Started Locally

You can run this project in two simple ways:

1. **Open directly in the browser**
   - Double-click `index.html` or open it in your preferred browser.

2. **Optional: Run with a simple static server**
   - Make sure you have Node.js installed.
   - In this folder, run:
     - `npm start`
   - This uses `npx serve .` to host files on a local HTTP server.

### Deploying to the Web

This is a static site, so you can host it almost anywhere. Popular options:

- **GitHub Pages**
- **Netlify**
- **Vercel**
- Any static hosting that serves `index.html`, `styles.css`, and `script.js`

See inline notes in this README or your hosting provider’s docs for exact steps.
